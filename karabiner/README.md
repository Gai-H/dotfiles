# Karabiner-Elements Configuration

## Aerospace

### Focus Mode

Hold `Option + F` to enter Focus Mode.

| Key | Action                    |
| :-- | :------------------------ |
| `h` | Focus window to the left  |
| `j` | Focus window down         |
| `k` | Focus window up           |
| `l` | Focus window to the right |

### Move Mode

Hold `Option + M` to enter Move Mode.

| Key | Action                   |
| :-- | :----------------------- |
| `h` | Move window to the left  |
| `j` | Move window down         |
| `k` | Move window up           |
| `l` | Move window to the right |

### Resize Mode

Hold `Option + R` to enter Resize Mode.

| Key | Action                       |
| :-- | :--------------------------- |
| `j` | Shrink window (smart resize) |
| `k` | Grow window (smart resize)   |

### Workspace Mode

Hold `Option + W` to enter Workspace Mode.

| Keys      | Modifier (Optional) | Action                                                                                     |
| :-------- | :------------------ | :----------------------------------------------------------------------------------------- |
| `1` - `0` | (None)              | Switch to the specified workspace.                                                         |
| `1` - `0` | `Shift`             | Move the focused window to the specified workspace **and switch focus** to that workspace. |

**Note:** Pressing `Option + W` by itself (without any other key) will switch you to the previously focused workspace.
